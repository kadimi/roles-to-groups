== Description ==

The "Automatic Roles to Groups for BuddyPress and BuddyBoss" plugin streamlines the process of assigning roles to groups within BuddyPress and BuddyBoss platforms. With this plugin, you can automatically assign users to groups based on their roles. This helps in managing group memberships more efficiently and ensures that users have the appropriate roles for their group activities.

== Installation ==

1. Download the plugin zip file.
2. Go to the WordPress admin area.
3. Navigate to Plugins > Add New.
4. Click on "Upload Plugin" and choose the zip file.
5. Click "Install Now" and then activate the plugin.

== Usage ==

1. Go to the "Roles to Groups" settings page.
2. Configure the plugin settings to define role-to-group mappings.
3. Save your settings. The plugin will now automatically assign roles to users based on your configurations.
